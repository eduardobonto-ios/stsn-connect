STSN Connect — System Overview

STSN Connect is a comprehensive, web-based school management system designed for St. Theresa's School Network. It integrates every aspect of school operations — from student admissions and grade encoding to payroll processing and parent communication — into a single, unified platform. The system supports both Basic Education (K–12) and College academic structures with a role-based, permission-driven architecture.

The Action Center is the unified approval queue that consolidates all pending workflow items across every module into one place. Staff can review, approve, or reject enrollment applications, discount requests, leave filings, assessment approvals, payroll runs, and more — all without navigating between modules.
🖥
Screenshot: Action Center
Showing: All Pending · Enrollments · Assessments · Discounts · Leave Requests · Payroll
Key Features
  • Unified queue for all pending approvals across modules
  • Filter by module type, date range, priority, and submitter
  • Bulk approve or reject multiple items simultaneously
  • Contextual detail panel with full request information
  • Add approval notes or rejection reasons before acting
  • Real-time badge count in sidebar showing pending items
  • Escalation indicators for items nearing SLA deadlines
  • Complete approval trail with who approved what and when
Workflow
  1. Staff navigates to Action Center from the sidebar
  2. System displays all pending items the user is authorized to approve
  3. User applies filters to narrow down by type or date
  4. User clicks a pending item to view the full detail panel
  5. User reviews the request, adds an optional note, then approves or rejects
  6. System sends the item to the next approval level or marks it complete
  7. Action is logged in the audit trail with timestamp and actor
Tabs / Views

The Enrollment module is the primary interface for processing student admissions and registrations each academic term. The registrar can search for existing students, process new enrollments through a multi-step wizard, review online applications from the queue, manage enrollment status, and generate the Certificate of Registration (COR).
🖥
Screenshot: Enrollment (Admission)
Showing: Admissions & Directory · Online Queue · Bulk Import
Key Features
  • Student search with filter by grade level, section, and enrollment status
  • KPI cards for enrolled count, pending/for-assessment count, and online application count
  • Multi-step Enrollment Wizard for complete student intake
  • Online application review queue for self-submitted enrollments
  • Bulk CSV import from the official registrar template
  • COR (Certificate of Registration) preview and print
  • Status filtering: Enrolled, Pending, For Assessment
  • Per-student enrollment status badge and quick-action buttons
Workflow
  1. Registrar navigates to the Enrollment module
  2. Reviews pending online applications from the Online Queue tab
  3. To enroll a new student: clicks 'New Enrollment' to open the Enrollment Wizard
  4. Wizard Step 1 — Personal Information: name, birthdate, LRN, contact details
  5. Wizard Step 2 — Guardian/Parent: parent name, relationship, contact
  6. Wizard Step 3 — Academic: grade level, section, subjects, payment terms
  7. Wizard Step 4 — Assessment & Fees: auto-computed tuition and miscellaneous fees
  8. Wizard Step 5 — Documents: attach enrollment requirements
  9. Registrar submits enrollment; status set to 'Pending' or 'For Assessment'
  10. Accounting approves the assessment; status changes to 'Enrolled'
  11. Registrar generates and prints the COR for the student
Tabs / Views
  • Admissions & Directory — Full list of students with enrollment status, KPI cards, and search/filter controls
  • Online Queue — Self-submitted enrollment applications from the Student Portal awaiting registrar review
  • Bulk Import — CSV masterlist upload tool with validation preview and import summary
